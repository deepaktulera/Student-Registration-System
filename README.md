# Student Registration System

It is a simple **Student Registration System** built using **HTML, CSS, and JavaScript**.

This project allows student to add, display, modify, and delete his data dynamically using **DOM Manipulation** and **Local Storage**.

## Features

* You can new student records
* Display registered students in a table
* Update/modify student details
* Delete student records
* Data stored in **Local Storage**
* Responsive design for mobile, tablet, and desktop

## Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (DOM Manipulation & Local Storage)**

## Project Files

* `index.html` → Main structure of the project
* `style.css` → Styling and responsive design for this project
* `script.js` → Functionality and DOM logic
* `README.md` → Project documentation

## How to Run the Project

1. Clone the project files from the git repositories or download the project files 
2. Clone that git link into vs code or open that downloaded folder in vs code
3. Run the `index.html` file in any web browser
